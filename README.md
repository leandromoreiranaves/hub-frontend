# HUB FRONTEND

Este repositório reúne **5 projetos desenvolvidos em React**, cada um localizado em uma pasta independente e com seu próprio ambiente de execução.

## Como utilizar

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git
```

### 2. Acesse o diretório do repositório

```bash
cd NOME-DO-REPOSITORIO
```

### 3. Escolha o projeto que deseja executar

Cada exercício está em uma pasta separada. Entre na pasta correspondente:

```bash
cd ex1
```

ou

```bash
cd ex2
```

ou

```bash
cd ex3
```

e assim por diante.

### 4. Siga as instruções do projeto

Cada pasta possui um arquivo **README.md** com as instruções específicas de instalação, dependências e execução.
Basta abrir o `README.md` da pasta escolhida e seguir os passos indicados.

## Observações

* Cada projeto é independente dos demais.
* As dependências devem ser instaladas dentro da pasta do projeto que será executado.
* Não é necessário instalar todas as aplicações para testar apenas uma delas.

## Tecnologias utilizadas

* React
* JavaScript / TypeScript
* Vite
* Axios
* Tailwind CSS
* Hooks
